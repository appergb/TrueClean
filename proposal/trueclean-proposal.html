01 / 看清楚

可视化磁盘占用

并行递归扫描整块硬盘或任意目录，把占用按 11 个类别拆开，两种可视化 + 文件树下钻。

  • Treemap 矩形树图 + Sunburst 旭日图
  • 11 类分类占比（系统/应用/开发/缓存/日志…）
  • 逐层下钻到最大子目录
  • 实时进度，可随时取消
02 / 清得安全

分级清理 + 回收站兜底

自动识别真正的"垃圾"，区分"绝对安全可删"和"需你确认"，删除默认进回收站。

  • 用户/系统/应用缓存、日志、临时文件
  • 浏览器缓存、开发缓存（DerivedData 等）
  • 包管理器缓存（npm/cargo/pip/gradle）
  • 默认走回收站，可恢复
03 / AI 帮判断

真正会用工具的 AI 助手

内置 AI Agent 面板，能调用扫描/清理工具基于真实数据给建议，绝不编造路径和体积。

  • 多模型可选（Claude/OpenAI/Ollama）
  • 工具调用 + 流式输出可视化
  • 按"可立即清理/建议复核/不要动"分组
  • 给出预计释放空间与风险等级

安全设计

安全第一，省空间第二

涉及删除的操作，每一步都把"安全"放在"省空间"之前。

✓ 绝对安全可删

用户缓存、系统缓存、应用缓存、各类日志、临时文件、浏览器缓存、开发缓存、回收站内容。删除后系统会自动重建，不丢失用户数据。

! 需用户确认

文档、媒体（照片/视频/音乐）、下载目录大文件、压缩包、应用程序本身、用户主目录文档区内容。删除前必须逐项征求确认。

✕ 绝不要动

macOS /System、/usr；Windows C:\Windows、System32；Linux /、/boot、/etc、/usr 等系统关键路径。AI 提示词设有安全红线，绝不建议删除。

🔒 兜底机制

删除默认走回收站（可恢复）；永久删除需显式选择；所有破坏性操作 UI 上需二次确认；API Key 仅保存在本地配置文件，不上传。
