## 一句话定位

**TrueClean 是一款开源、跨平台、内置 AI Agent 的桌面磁盘清理工具——看得清空间被谁吃掉，删得安全可撤销，让 AI 基于真实数据帮你判断什么能清。**

类 CleanMyMac，但开源、跨三平台、AI 真正干活、安全逻辑可审计。

---

## 问题与市场

### 痛点

硬盘满了是普遍痛点，但现有方案各有缺陷：

| 方案 | 缺陷 |
|---|---|
| 系统自带工具 | 粒度粗、可视化弱、无 AI |
| CleanMyMac | 仅 macOS、年订阅 ¥300+、闭源不可审计 |
| DaisyDisk | 仅 macOS、只看不能清、无 AI |
| BleachBit | 界面陈旧、无可视化、无 AI、跨平台体验割裂 |

### 市场规模

- **Mac 清理软件市场**：CleanMyMac 年收入过亿美元，证明付费意愿存在。
- **开发者群体**：全球 3000 万+ 开发者，开发缓存（DerivedData/node_modules/cargo）是高频清理需求，无专门工具。
- **AI 桌面助手趋势**：2024-2026 是 AI Agent 落地桌面的一年，"AI + 工具调用"范式成熟。

### 目标用户

技术开发者（核心）+ 内容创作者 + 普通用户。先打开发者社区（GitHub 传播），再扩展到大众。

## 差异化（为什么是 TrueClean）

### 三个核心壁垒

1. **AI Agent 真正干活**
   - 不是聊天机器人。Agent 可调用 9 个真实工具（扫描目录、找垃圾、找重复、列应用、清理路径…），基于**用户磁盘的真实数据**给建议，并在用户确认后执行。
   - 多 Provider（Claude/OpenAI/Ollama），BYOK 零订阅成本，Ollama 本地模型隐私优先。

2. **安全可审计、可撤销**
   - 删除逻辑全开源，`is_protected` 保护路径表硬编码可审计。
   - 默认进回收站 + `CleanManifest` 快照 + 一键 `restore_last` 撤销。
   - 竞品要么闭源不可审计（CleanMyMac），要么无撤销（BleachBit）。

3. **跨平台 + 开发者友好**
   - 一套 Rust + React 代码三平台（macOS/Windows/Linux），Tauri 2 安装包仅 ~10MB。
   - 专门识别开发缓存（Xcode DerivedData、node_modules、npm/cargo/pip/gradle 缓存）。
   - blake3 内容哈希去重、Treemap/Sunburst 可视化下钻。

### 竞品对比一句话

> CleanMyMac 收钱且只支持 Mac；BleachBit 免费但没 AI 没可视化没撤销；DaisyDisk 只看不能清。TrueClean 免费、跨平台、有 AI、可撤销、安全可审计。

## 技术壁垒

| 壁垒 | 说明 |
|---|---|
| **Tauri 2 + Rust 内核** | 并行扫描（jwalk/rayon）、blake3 哈希、跨平台回收站（trash crate），性能与安全兼得。竞品多为 Electron 或原生单平台。 |
| **中立 Provider 架构** | 工具 schema 定义一次，各 Provider 适配层转换。新增 AI 厂商只加一个文件，不锁定。 |
| **安全红线 + 撤销机制** | `is_protected` + `CleanManifest` + `restore_last` 是经过测试的信任基石，非一日之功。 |
| **数据契约冻结** | `model.rs` ↔ `types.ts` 单一真源，多 Agent 并行开发不冲突，工程化程度高。 |

## 商业模式

- **开源免费**（MIT），无订阅。
- **AI 能力 BYOK**：用户自带 API Key，TrueClean 不代售、不抽成。
- **变现路径（未来）**：可选的托管 AI 服务（一键开通，按量计费）、企业版（集中配置/审计）。
- **核心价值不在卖软件，而在建社区与信任**：开源安全逻辑 = 用户敢用 = 口碑传播。
